# データ整理

## train
- PassengerId – 乗客識別ユニークID
- Survived – 生存フラグ（0=死亡、1=生存）
- Pclass – チケットクラス
  - 1: 上層クラス（お金持ち）
  - 2: 中層クラス（一般階級）
  - 3: 下層クラス（労働階級）
- Name – 乗客の名前
- Sex – 性別（male=男性、female＝女性）
- Age – 年齢
- SibSp – タイタニックに同乗している兄弟/配偶者の数
- Parch – タイタニックに同乗している親/子供の数
- Ticket – チケット番号
- Fare – 料金
- Cabin – 客室番号
- Embarked – 出港地（タイタニックへ乗った港）
  - C: Cherbourg
  - Q: Queenstown
  - S: Southampton
